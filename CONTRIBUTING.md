Thanks for your interest in Survive2Sunrise!

This project is a C++ learning exercise first — a small terminal game used to practice object-oriented design, game loops, and ncurses. Contributions that keep the codebase readable and focused are welcome.

## Getting started

1. Fork the repository and create a branch from `main`.
2. Build from source (see [README.md](README.md#build-from-source)).
3. Make your changes in a focused commit or pull request (one feature or fix at a time).
4. Open a pull request with a short description of what changed and why.

## Guidelines

- Match the existing code style and file layout (`src/` modules, CMake build).
- Keep changes scoped — avoid unrelated refactors in the same PR.
- If you add a feature, update the README or player docs when behavior or controls change.
- Bug reports and feature ideas are welcome via [GitHub Issues](https://github.com/MahdiT54/fnaf-cpp/issues).

## Questions

If something in the build steps or project layout is unclear, open an issue. That helps future contributors too.
